There are several different python entrypoint routines to test

a) publish_random_number.py:
	simple publish of a random integer generated inside the docker container and to publish the number on http://repi.record-evolution.com/

b) turn_on_led.py: 
	remote procedure call to turn on an led by clicking a button on the homepage http://repi.record-evolution.com/

c) DHT11.py:
	publish a temperature signal on the homepage http://repi.record-evolution.com/

d) subscribe to an event and output its string to the digital screen connected to the raspberry pi

e) push a button and publish the event to the homepage http://repi.record-evolution.com/
